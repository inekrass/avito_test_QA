# Avito.Tech Test Project QA

Тестовое задание от **Avito.Tech** на должность QA-инженера.

## 📁 Структура проекта

```
avito_test_QA/
├── README.md                          # Этот файл (общее описание)
├── t-1/                               # Первое тестовое задание (документация)
│   └── README.md
├── t-2.2/                             # Второе тестовое задание (автотесты)
│   ├── README.md                      # Документация по запуску
│   ├── BUGS.md                        # Найденные баги
│   ├── TESTCASES.md                   # Описание тест-кейсов
│   ├── conftest.py                    # pytest конфиг
│   ├── requirements.txt               # Зависимости
│   ├── .gitignore
│   ├── pages/                         # Page Object Model для страниц
│   └── tests/                         # Автотесты  


```
