# Баг-репорт 1

|Заголовок | [bug][web][prod] Отображаемые объявления не соответствуют указанному диапазону цен |
|:----------|:----------------------------------------------------------------------------------------------------------------|
| Описание | При указании диапазона цен с валидными значениями (От - 1000; До - 50000) программа выдает объявления, которые не находятся в указанном диапазоне |
| Предусловия | Фильтры не выставлены |
| Шаги воспроизведения| **1.** Ввести валидное (>=0) минимальное значение цены = 1000.<br>**2.** Ввести валидное (>=0) максимальное значение цены = 50000. |
| Текущий результат | **1.** Фильтр применяется.<br>**2.** В список попадают объявления, которые выходят за рамки указанного диапазона цен. |
| Ожидаемый результат | **1.** Фильтр применяется.<br>**2.** Программа отображает только объявления, которые подходят под условия указанного диапазона |
| Окружение | 1. "Модерация Авито" WEB<br>2. Браузер Google Chrome Версия: 146.0.7680.155 (официальная сборка) (arm64) |
| Тестовые устройства | Macbook Pro 16 M1 Pro macOS Tahoe 26.1 (25B78)|
| Priority | High |
| Severity | High |
| Логи | ```AssertionError: BUG: Найдены объявления с ценами вне диапазона 1000-50000. Проблемные цены: [54487, 50162, 53060, 52311, 54377, 50402, 54459, 54194, 51536]``` |
