## Страница статистики (Web)
|Название             | Обновить статистику |

| Приоритет           | High |
| Предусловие         | - |
| Шаги                | 1. Зафиксировать текущее состояние статистики.<br>2. Внести изменения - Одобрить объявление<br>3. Нажать кнопку "Обновить" |
| Ожидаемый результат | Данные обновляются и progress bar таймера сбрасывается |
| Постусловие         | - |

---

|Название             | Остановить таймер |
|:---------------------|:------------------------|
| Приоритет           | High |
| Предусловие         | - |
| Шаги                | 1. Зафиксировать текущее состояние статистики.<br>2. Внести изменения - Одобрить объявление<br>3. Нажать кнопку остановки таймера<br>4. Выждать 5 минут, чтобы убедиться, что по их истечении статистика не обновится. |
| Ожидаемый результат | 1. Автоматическое обновление останавливается.<br>2. Статистика не обновляется по истечении 5 минут.<br>3. Progress bar таймера останавливается |
| Постусловие         | - |

## Mobile

| Постусловие         | - |
